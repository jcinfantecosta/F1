# F1 Racing Game

Juego de carreras de Fórmula 1 desarrollado en Python con Pygame.
El jugador debe esquivar obstáculos mientras avanza; la velocidad
aumenta en cada nivel, incrementando la dificultad.

## Tecnologías
- Python
- Pygame

## Cómo ejecutarlo
pip install pygame
python juego.py
